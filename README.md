Firejail Fedora role
====================

Deploys Firejail on a Fedora workstation


Requirements
------------

  * Fedora workstation


Role Variables
--------------


Dependencies
------------

  * python-dnf


Example Playbook
----------------

```
- name: Install Firejail
  hosts: localhost
  roles:
    - gbraad.firejail
```
